Leafy Green Notes

Leafy Green Notes is a React App project that I created in preparation for an interview for the design systems engineer position at MongoDB. I implemented MongoDB's Leafy Green UI components and used MongoDB Realm as my data store.

My main goal for this project was to get familiar with MongoDB's UI system. I wanted to bring my perspective to the interview on what I liked and what I felt was missing/could use improvement. This was a great opportunity to show my interest and readiness for the position.


After completing this goal, I decided to integrate MongoDB's Realm service. I implemented user authentication and storage. This was my first time integrating a managed database and user authentication. It proved to be a fun learning experience!
